#Modelo Termal para el margen Andino
#Basado en los datos Tassara et al.
#Lat 18 S hasta 45 S
#Enero 2015
#---------------------------------------------------------------------------------
# Parametros para generar Modelo Termal
#---------------------------------------------------------------------------------
#    Valor  Nombre      Descripcion
#    -----  ----------  -----------
     False  k_z         k_z = False prom. de K.
#                       k_z = True  valor de K dep de Z
     False  H_z         H_z = False prom. de H.
#                       H_z = True  valor de H dep de Z
     False  delta_icd   delta_icd = True delta depende de icd.
#                       delta_icd = False => delta_icd = delta
      True  t_lat       t_lat = True usar edad de la fosa segun latitud.
#                       t_lat = False (t_lat = t)
#·········  ··········  ··························································
       3.0  k_cs        Conductividad termica Corteza superior [W/mK]
       3.0  k_ci        Conductividad termica Corteza inferior [W/mK]
       3.0  k_ml        Conductividad termica Manto litosferico [ W/mK]
      3e-6  H_cs        Calor radiogenico Corteza superior [W/m3]
      3e-6  H_ci        Calor radiogenico Corteza inferior [W/m3]
      3e-6  H_ml        Calor radiogenico Manto litosferico [W/m3]
    1.0e-6  kappa       difusividad termica [m2/s]
    1250.0  Tp          Temperatura potencial del manto [C]
      4e-4  G           Gradiente adiabatico [K/m]
     6.5e4  V           Velocidad de convergencia [m/Myr]
       1.0  b           Parametro adimensional
      20.0  dip         Angulo de subduccion
      1e-3  D           Constante de proporcionalidad adimensional
        10  delta       Decaimiento radiogenico (km)
        30  t           Edad promedio de la placa oceanica en la fosa (Ma)
